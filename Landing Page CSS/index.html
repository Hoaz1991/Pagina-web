¿que es una pagina web?

es un documento o información electrónica capaz de contener texto, sonido, vídeo, programas, enlaces, imágenes, hipervínculos y muchas otras cosas, adaptada para la llamada World Wide Web, y que puede ser accedida mediante un navegador web.

HTML, CSS y JavaScript. Lenguajes para el desarrollo de páginas web

Estos son los tres lenguajes principales que se utilizan en el front-end de nuestra aplicación, es decir, se utilizan en el lado del cliente en el propio navegador.

click para mas informacion

Mi nombre es HTML

soy el lenguaje de marcado utilizado para estructurar elementos de la página, como párrafos, enlaces, títulos, tablas, imágenes e incluso videos.

Mi nombre es CSS

soy el lenguaje de estilo utilizado para definir colores, fuentes, tamaños, posicionamiento y cualquier otro valor estético para los elementos de la página.

Mi nombre es JS

lenguaje de programación utilizado para hacer la página con más movimiento, pudiendo actualizar elementos de forma dinámica y manejar mejor los datos enviados y recibidos en la página.

Preguntas frecuentes

¿como crear un sitio web con html?

Los documentos HTML son archivos con extensiones .html o .htm y pueden ser visualizados en cualquier navegador. El cual hace la lectura del archivo y renderiza el contenido para que los usuarios puedan verlo.

¿por que implementar css en un sitio web?

En lo relativo a páginas web, no solo cuenta el contenido, sino también su presentación. El lenguaje de hojas de estilo CSS se utiliza habitualmente para dar un formato adecuado al HTML. De este modo, se definen atributos como el diseño, el color y la forma de los elementos individuales de HTML. Las instrucciones de diseño CSS existen independientemente de HTML y también deben integrarse en un documento electrónico.

¿es importante javascript para tu pagina web?

es importante javascript para que puedas empezar añadiendole pequeñas cosas como carruseles, galerías de imágenes, diseños fluctuantes, y respuestas a las pulsaciones de botones. Con más experiencia, serás capaz de crear juegos, animaciones 2D y gráficos 3D, aplicaciones integradas basadas en bases de datos ¡y mucho más!
